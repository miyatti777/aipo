# 02 発見: ペルソナ作成

## 最初に質問（実行前に回答してください）

- プロジェクト名（必須）
- ターゲットユーザーの基本情報（必須）
- ペルソナの詳細設定（行動パターン・ニーズ・課題等）（必須）
- 参考となる調査データ（任意）

## 目的

Rules（`basic/02_pmbok_discovery.mdc`）に準拠し、プロジェクトのターゲットユーザーを具体化したペルソナを作成します。顧客調査の結果を活用して、リアルで実用的なペルソナを生成します。

## 必要入力

- プロジェクト名: 対象プロジェクトの名称
- 基本情報: 年齢・性別・職業・居住地等の基本属性
- 行動パターン: 日常の行動・利用シーン・購買行動
- ニーズ・課題: 抱えている問題・求めている価値
- 参考データ: 顧客調査結果・市場データ等（任意）

## 実行手順（Rules Steps）

```yaml
- trigger: "(ペルソナ作成|Persona)"
  priority: high
  steps:
    - call basic/02_pmbok_discovery.mdc => persona_questions
    - wait_for_all_answers
    - confirm: "ペルソナを作成します。よろしいですか？"
    - create_markdown_file:
        path: "{{patterns.draft_persona}}"
        template_reference: "02_pmbok_discovery.mdc => persona_template"
```

## ペルソナ構成要素

### 📊 基本情報

- *名前: 覚えやすい仮名*
- *年齢・性別: 具体的な属性*
- *職業・役職: 社会的ポジション*
- *居住地: 地理的情報*
- *家族構成: ライフスタイルに影響する要素*

### 🎯 行動・心理

- *日常の行動パターン: 1日の過ごし方*
- *情報収集方法: メディア・SNS・口コミ等*
- *意思決定プロセス: 購買・選択の基準*
- *価値観: 重視すること・優先順位*

### 💡 ニーズ・課題

- *現在の課題: 抱えている問題・不満*
- *求めている価値: 欲しい解決策・体験*
- *利用シーン: いつ・どこで・なぜ使うか*
- *期待する成果: 理想的な結果*
