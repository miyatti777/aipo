# CMD_creative_パーティー企画

最終更新日時: 2026年1月20日

## 📋 コマンド概要

```yaml
command: /creative/パーティー企画
alias: ["/party", "/イベント企画", "/お祝い"]
category: creative
related_role: party_planner_focus

description: |
  パーティー・お祝いイベントの企画をサポートします。
  誕生日、送別会、歓迎会、サプライズなど
  思い出に残るイベントを設計。

trigger_keywords:
  - "パーティー"
  - "誕生日"
  - "送別会"
  - "歓迎会"
  - "サプライズ"
```
