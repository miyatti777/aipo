# CMD_creative_物語執筆

最終更新日時: 2026年1月20日

## 📋 コマンド概要

```yaml
command: /creative/物語執筆
alias: ["/story", "/小説", "/執筆"]
category: creative
related_role: creative_writer_focus

description: |
  小説・物語の企画から執筆までをサポートします。
  コンセプト設計、キャラクター作成、プロット構成、
  執筆・推敲の全工程を支援。

trigger_keywords:
  - "小説"
  - "物語"
  - "執筆"
  - "書く"
  - "創作"
```
