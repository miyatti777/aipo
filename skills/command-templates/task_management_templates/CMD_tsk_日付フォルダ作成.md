# CMD-02: 日付フォルダ作成コマンド

## 📌 コマンド概要

**目的**: Flow配下に指定月の日付フォルダを一括作成

**用途**: 月初や途中参加時に、その月の日付フォルダをまとめて作成

**実行時間**: 30秒以内

---

## 📥 Input

### 必須項目

- **年月** (オプション): 例）202512、2025-12
    - 未指定の場合は現在の年月を使用

### 入力例

```jsx
@CMD-02_日付フォルダ作成 を実行してください

年月: 202512
```

または：

```jsx
@CMD-02_日付フォルダ作成 を実行してください
```

（現在の年月で実行）

---

## 📤 Output

### 作成されるフォルダ構造

```jsx
Flow/
└── 202512/ 📅
    ├── 2025-12-01/ 🗓️
    ├── 2025-12-02/ 🗓️
    ├── 2025-12-03/ 🗓️
    ├── ...
    └── 2025-12-31/ 🗓️
```

### 処理内容

- ✅ その月の日数分のフォルダを作成（28～31日）
- ✅ 既に存在する日付フォルダはスキップ
- ✅ 不足している日付フォルダのみ追加

---

## 🔧 実行方法

AIアシスタントに以下のように依頼してください：

```
CMD-02_日付フォルダ作成 を実行してください

年月: 202512
```

または：

```
今月の日付フォルダを作成してください
```

---

## 📋 処理ロジック

### ステップ1: パラメータ確認

1. 年月が指定されているか確認
2. 未指定の場合は現在の年月を取得
3. 年月の形式を正規化（YYYYMM形式）

### ステップ2: Flow構造の確認

1. Flow/YYYYMMフォルダの存在確認
2. 存在しない場合は作成
3. 既存の日付フォルダを確認

### ステップ3: 日付フォルダの作成

1. その月の日数を計算（うるう年対応）
2. 1日から月末まで順次処理：
    - フォルダ名: YYYY-MM-DD
    - アイコン: 🗓️
    - 親: Flow/YYYYMM
    - 既に存在する場合はスキップ
3. 作成した日付フォルダ数をカウント

### ステップ4: 完了メッセージ

```jsx
✅ 日付フォルダ作成が完了しました！

対象月: 2025年12月
作成済み: 15個
新規作成: 16個
合計: 31個

Flow/202512/ 配下に全日付フォルダが準備されました。
```

---

## ⚠️ 注意事項

### エラーハンドリング

1. **Flowフォルダが存在しない場合**
    - エラーメッセージ: 「Flowフォルダが見つかりません。先にインストールコマンドを実行してください。」
2. **不正な年月形式の場合**
    - エラーメッセージ: 「年月の形式が正しくありません。YYYYMM形式で指定してください。例: 202512」
3. **権限エラーの場合**
    - エラーメッセージ: 「フォルダ作成に失敗しました。権限を確認してください。」

### 実行時間

- 目標: 30秒以内
- 月末日数31日の場合でも30秒以内に完了

### 使用シーン

- 月初に実行して1ヶ月分を一括準備
- 途中参加時に過去の日付フォルダを補完
- 他の月のフォルダを事前準備
