# 07 管理: 日次タスク作成

## 最初に質問（実行前に回答してください）

- 特になし（自動実行可能）

## 目的

日次タスクファイルを作成し、今日実行すべきタスクを整理します。バックログやカレンダーと連携して、効率的なタスク管理を実現します。

## 必要入力

- なし（自動実行）または以下を任意で指定
- 対象日（デフォルト：今日）
- 対象プロジェクト（デフォルト：全プロジェクト）

## 機能詳細

### 🚀 タスク収集

- **バックログ連携**: 進行中のストーリー・タスクを抽出
- **カレンダー統合**: 本日の予定を取り込み
- **ルーチンタスク**: 定期的なタスクを自動追加

### 📋 生成内容

- 日次タスクリスト
- カレンダー予定
- 進捗管理

### 📁 保存場所

```
[project]/daily/[YYYY-MM-DD]/
├── daily_tasks.md          # 日次タスクリスト
├── calendar_events.md      # カレンダー予定（任意）
└── notes.md                # 作業メモ（任意）
```

## 日次タスクテンプレート

```markdown
# 日次タスク（[YYYY-MM-DD]）

## 📅 本日の予定

| 時間 | 予定 | 備考 |
|------|------|------|
| 09:00-10:00 | [予定1] | |
| 10:00-12:00 | [予定2] | |
| 14:00-15:00 | [予定3] | |

## ✅ 今日のタスク

### 🔴 最優先（Must）
- [ ] [タスク1]
- [ ] [タスク2]

### 🟡 重要（Should）
- [ ] [タスク3]
- [ ] [タスク4]

### 🟢 できれば（Could）
- [ ] [タスク5]

## 📝 作業ログ

### [HH:MM] [作業内容]
- メモ：[詳細]

## 🔄 明日への引き継ぎ
- [ ] [引き継ぎタスク1]
- [ ] [引き継ぎタスク2]

## ふりかえり
- **良かったこと**: 
- **改善点**: 
- **学び**: 
```

## 生成物

- `daily_tasks.md`（日次タスク）

## 次に実行

- タスク完了後、チェックボックスを更新
- 1日の終わりに「ふりかえり」を記入
- 翌日の準備として「明日への引き継ぎ」を更新
