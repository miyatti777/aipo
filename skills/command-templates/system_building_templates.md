# System Building Templates

システム設計・実装のためのコマンドテンプレート集

## 概要

IDE（Cursor/Claude Code）環境でのシステム開発をサポートするテンプレートです。
アーキテクチャ設計から本番デプロイまで、一貫した品質でシステムを構築できます。

## 対応技術スタック

### フロントエンド
- Next.js 14+ (App Router)
- React 18+
- Tailwind CSS
- shadcn/ui

### バックエンド
- Next.js API Routes / Server Actions
- Prisma ORM
- PostgreSQL

### インフラ
- Vercel
- GitHub Actions
- Sentry

## 使い方

```

システム名: [あなたのシステム名]
目的: [解決したい課題]
```
