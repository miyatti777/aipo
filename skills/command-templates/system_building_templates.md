# 概要

システム・業務基盤の構築に関するテンプレート群です。「動くDB/システム」を作ることを目的とします。

## 対象範囲

- Database設計・構築
- 業務プロセス設計
- 運用ルール定義
- Wiki構築・ドキュメント管理
