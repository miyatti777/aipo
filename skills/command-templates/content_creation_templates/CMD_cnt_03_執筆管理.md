# Phase 3: 執筆の実施とサポート

## 🎯 実行内容

実際に執筆を進める。AIがサポート可能な場合はサポートする。

### 3.1 執筆サポート

**AIができるサポート：**

1. **アウトライン確認**
    - セクションごとの要点を再確認
    - 書くべき内容のリマインド
2. **執筆ブロック解消**
    - 書けない時の相談
    - 別の切り口の提案
    - 具体例の提案
3. **文章の推敲**
    - わかりにくい箇所の改善案
    - 冗長な表現の削減
    - 論理構成の確認
4. **進捗確認**
    - 現在の文字数
    - 残りのセクション
    - 期限までのペース

### 3.2 執筆の進め方

**推奨ステップ：**

1. **セクションごとに執筆**
    - アウトラインを見ながら1セクションずつ
    - 完璧を目指さず、まず下書きを完成させる
2. **執筆後に推敲**
    - 全体の流れを確認
    - 重複や矛盾をチェック
    - 文章を磨く
3. **文字数調整**
    - 目標文字数と比較
    - 必要に応じて加筆・削減

# Phase 4: 推敲と調整

## 🎯 実行内容

記事全体を推敲し、品質を高める。

### 4.1 推敲チェックリスト

**確認項目：**

### 構成・論理

- [ ]  イントロで興味を引けているか
- [ ]  セクション間の繋がりが自然か
- [ ]  主張に根拠があるか
- [ ]  結論が明確か

### 文章・表現

- [ ]  わかりやすい表現になっているか
- [ ]  冗長な部分はないか
- [ ]  専門用語に説明があるか
- [ ]  例が適切か

### ターゲット読者

- [ ]  読者の課題に応えているか
- [ ]  読者の前提知識に合っているか
- [ ]  読後に期待する行動を促せているか

### 文字数・体裁

- [ ]  目標文字数の範囲内か（±10%程度）
- [ ]  見出しレベルが適切か
- [ ]  リストや図表が効果的に使われているか

### 4.2 推敲の実施

**推敲方法：**

1. **全体を通読**
    - 読者視点で読む
    - 違和感のある箇所をメモ
2. **セクションごとに推敲**
    - 論理の流れを確認
    - 文章を磨く
    - 不要な部分を削除
3. **最終確認**
    - もう一度全体を通読
    - タイトルと内容が合っているか
    - コアメッセージが伝わるか

## 💡 執筆のコツ

### 執筆ブロックを防ぐ

1. **完璧を目指さない**
    - まず下書きを完成させる
    - 推敲は後で
2. **書きやすいところから**
    - 順番通りでなくてOK
    - 書けるセクションから着手
3. **小さく区切る**
    - 1セクション = 1タスク
    - 達成感を積み重ねる

### 執筆ペースを保つ

1. **毎日少しずつ**
    - 短時間でも継続
    - 習慣化が重要
2. **集中時間を確保**
    - 通知をオフ
    - 執筆に集中できる環境
3. **進捗を可視化**
    - 文字数カウント
    - 完了セクション数
