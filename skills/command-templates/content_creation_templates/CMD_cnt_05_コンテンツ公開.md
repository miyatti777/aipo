# Phase 3: 記事公開

## 🎯 実行内容

記事を公開し、プロモーションを実施する。

### 3.1 公開手順

**ステップ：**

1. **最終確認**
    - 公開前に記事を一度通読
    - プレビューで表示を確認
2. **公開実行**
    - 公開ボタンを押す
    - 公開URLを確認
    - 実際の表示を確認
3. **公開URLの記録**
    - 公開URLをメモ
    - プロモーション素材に反映

### 3.2 プロモーション実施

**実施順序：**

1. **即時投稿（公開直後）**
    - Twitter/X
    - 自社SNSアカウント
2. **時間差投稿（30分〜1時間後）**
    - LinkedIn
    - Facebook
3. **コミュニティ共有（1〜2時間後）**
    - 関連Slackチャンネル
    - オンラインコミュニティ
    - メーリングリスト

### 3.3 プロモーション記録Database の作成

**必須プロパティ：**

| プロパティ名 | 型 | 説明 |
| --- | --- | --- |
| プロモーション先 | title | SNSやコミュニティ名 |
| 投稿日時 | date | 投稿した日時 |
| 投稿内容 | text | 投稿した文章 |
| 投稿URL | url | 投稿のURL |
| ステータス | status | 予定/投稿済み/未投稿 |
| 反応 | text | いいね数、リツイート数等 |
| メモ | text | 気づいた点 |

# Phase 4: 効果測定

## 🎯 実行内容

公開後の効果を測定し、記録する。

### 4.1 効果測定Database の作成

**必須プロパティ：**

| プロパティ名 | 型 | 説明 |
| --- | --- | --- |
| 測定日 | date | 測定した日 |
| 経過日数 | formula | 公開からの経過日数 |
| PV | number | ページビュー |
| いいね | number | いいね数 |
| シェア | number | シェア数 |
| コメント | number | コメント数 |
| 滞在時間 | text | 平均滞在時間 |
| 参照元 | text | 主な流入元 |
| メモ | text | 気づいた点 |

### 4.2 測定タイミング

**定期測定：**

1. **公開直後（〜6時間）**
    - 初動の反応を確認
    - プロモーションの効果確認
2. **公開後24時間**
    - 1日の総PVを確認
    - SNSの反応を集計
3. **公開後1週間**
    - 短期的な効果を評価
    - コメントやフィードバックを確認
4. **公開後1ヶ月**
    - 中長期的な効果を評価
    - 検索流入を確認

### 4.3 分析観点

**確認すること：**

- **数値面**
    - 目標達成度
    - プラットフォーム別の反応
    - 時間帯別の流入
- **質的面**
    - コメントの内容
    - 読者の反応
    - 想定外の気づき
- **改善点**
    - 次回に活かせること
    - タイトルや導入の効果
    - プロモーション方法の有効性
