# CMD_biz_業務分析

最終更新日時: 2026年1月20日

## 📋 コマンド概要

```yaml
command: /biz/業務分析
alias: ["/ba", "/業務フロー", "/as-is-to-be"]
category: business_analysis
related_role: business_analyst_focus

description: |
  業務プロセスの分析と改善提案を行います。
  As-Is → To-Be → Gap分析の流れで
  業務の可視化から改善計画まで支援。

trigger_keywords:
  - "業務分析"
  - "業務改善"
  - "プロセス"
  - "フロー"
  - "要件定義"
```
