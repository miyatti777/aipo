# 04 実行: 議事録

## 最初に質問（実行前に回答してください）

- 会議名・目的（必須）
- 開催日時・場所（必須）
- 参加者（必須）
- 議題・アジェンダ（必須）
- 決定事項・アクションアイテム（必須）

## 目的

PMBOK実行フェーズの手法に基づき、プロジェクト関連会議の議事録を作成します。会議の内容を正確に記録し、決定事項とアクションアイテムを明確化します。

## 必要入力

- 会議名: 会議の名称・目的
- 開催情報: 日時・場所・形式（対面/オンライン）
- 参加者: 出席者・欠席者・役割
- 議題: 討議内容・アジェンダ
- 決定事項: 会議で決まったこと
- アクションアイテム: 誰が・何を・いつまでに

## 議事録の構成要素

### 📋 基本情報

| 項目 | 内容 |
|------|------|
| **会議名** | 会議の正式名称 |
| **目的** | 会議の目的・ゴール |
| **日時** | 開催日時・所要時間 |
| **場所** | 開催場所・形式 |
| **参加者** | 出席者・欠席者・役割 |

### 📝 会議内容

- **議題**: 討議したトピック・アジェンダ
- **討議内容**: 主要な発言・意見・議論
- **資料**: 使用した資料・参考文書
- **質疑応答**: 質問と回答の内容

### ✅ 決定事項・アクション

- **決定事項**: 会議で決まったこと・合意事項
- **アクションアイテム**: 具体的な行動項目
- **担当者**: 各アクションの責任者
- **期限**: 完了予定日・マイルストーン
- **次回会議**: 次回の予定・議題

## 議事録テンプレート

```markdown

## 参加者

| 名前 | 役割 | 出欠 |
```

## 議事録のベストプラクティス

### ✅ 良い議事録

- **客観的**: 事実を客観的に記録
- **具体的**: 曖昧さのない具体的な記述
- **行動指向**: アクションアイテムが明確
- **追跡可能**: 後で確認・フォローできる

### ❌ 避けるべき議事録

- **主観的**: 個人の感想・解釈が混在
- **曖昧**: 「検討する」「考慮する」等の曖昧な表現
- **行動不明**: 誰が何をするか不明確
- **期限なし**: いつまでにやるか不明
