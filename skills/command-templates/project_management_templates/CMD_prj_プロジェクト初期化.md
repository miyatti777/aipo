# CMD-03: プロジェクト初期化コマンド

## 📌 コマンド概要

**目的**: PMBOK構造のプロジェクトフォルダを作成

**用途**: Part2-2で使用（デモ、5分、座学内）

**実行時間**: 1分以内

---

## 📥 Input

- **プロジェクト名** (必須): 例）「春の新生活特集」「夏のアウトドアガイド」
- **プロジェクト目的** (オプション): 例）「新生活を始める読者向けのおすすめ商品紹介」
- **作成場所** (オプション): デフォルトはStock/programs配下

---

## 📤 Output

### 作成されるフォルダ構造

```
Stock/Programs/[Program名]/[プロジェクト名]/
├── README
├── 01_meetings/
│   ├── logs/（議事録）
│   └── summary/（サマリー）
├── 02_documents/
│   ├── 企画/
│   ├── 仕様書/
│   └── ガイドライン/
└── 03_tasks/
    └── タスク管理（Database）
```

### 記事制作向けカスタマイズ

- ✅ 記事制作に特化したフォルダ名
- ✅ キーワード戦略・競合分析テンプレート
- ✅ 記事リストDatabase（タイトル、ステータス、担当者、公開日）
- ✅ 執筆スケジュールテンプレート

---

## 🔧 実行方法

Notion AIチャットで以下のように実行してください：

```jsx
@CMD-03_プロジェクト初期化 を実行してください
```

## ⚠️ 注意事項

### エラーハンドリング

1. **プロジェクト名が未入力の場合**
    - エラーメッセージ: 「プロジェクト名を入力してください。」
    - デフォルト値: 「新規プロジェクト_[日付]」
2. **Stock/programsフォルダが存在しない場合**
    - 警告メッセージ: 「Stock/programsフォルダが見つかりません。先にインストールコマンドを実行してください。」
    - または、指定された場所に作成
3. **同名のプロジェクトフォルダが存在する場合**
    - 警告メッセージ: 「同名のプロジェクトが既に存在します。別の名前を指定してください。」
